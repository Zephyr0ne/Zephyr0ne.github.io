如何创建个人第一台云服务器

发表于 2022-06-07 阅读次数：

部署MySQL(保姆级教程)

前置任务

  1. 点击腾讯云网站
  2. 注册登录账号进行购买轻量云服务器
  3. 进入云服务器控制台重置初始密码
  4. 防火墙设置：22端口访问权限设置为本机IP（防止恶意攻击）

安装docker

  1. 使用Xshell连接工具，新建会话

  2. 登录账号之后首先更新yum 终端输入 yum update

    出现上面的complete说明更新成功

  3. 设置Docker yum源(yum是Linux系统的安装软件必备工具)
    sudo yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
    如果出现报错信息

    则点击 centOS8 yum源更换镜像 查看解决方法
    如果还是有错误信息则输入
    yum -y install yum-utils yum clean all yum makecache
    来安装yum工具类

  4. 查看当前可以安装docker-ce版本
    yum list docker-ce --showduplicates | sort -r

  5. 安装指定版本的格式 ,注意3:xxx 请移除3:
    yum -y install docker-ce-[VERSION]
    安装成功之后

  6. 重启Docker
    systemctl restart docker

  7. 设置Docker开机启动
    systemctl enable docker

  8. 查看Docker版本信息
    docker version

docker已经成功安装！

使用docker部署MySQL镜像

  1. 拉取Mysql镜像
    docker pull mysql:8.0.25
  2. 服务器上创建对应的数据目录
    mkdir -p /data/scms/mysql8/{logs,conf,data}
    目录说明：
    logs为日志目录，conf为my.cnf配置文件目录，data为数据文件存放路径
  3. 执行容器创建命令：
    ```
    docker run -p 3306:3306 -d –name mysql \

–privileged=true
-v /data/rushB/MySQL8/data:/var/lib/mysql
-v /data/rushB/MySQL8/conf:/etc/mysql
-v /data/rushB/MySQL8/logs:/var/log/mysql
-v /etc/localtime:/etc/localtime
-e MYSQL_ROOT_PASSWORD=密码
mysql:8.0.25

部署完之后输入  
`docker ps`  
查看mysql容器状态
![](https://zephyr0ne-1311513069.cos.ap-chengdu.myqcloud.com/%E5%BE%AE%E4%BF%A1%E5%9B%BE%E7%89%87_20220607155713.png)  
4. 进入服务器控制台添加防火墙规则  
![](https://zephyr0ne-1311513069.cos.ap-chengdu.myqcloud.com/%E5%BE%AE%E4%BF%A1%E5%9B%BE%E7%89%87_20220607160242.png)
5. 使用Navicat新建MySQL连接。输入账号密码，测试连接，显示连接成功！  
PS：如果第5步连接不成功，去Xshell终端输入  
`firewall-cmd --list-all`  
查看端口是否存在  
如果没有3306端口则输入  
`firewall-cmd --zone=public --add-port=3306/tcp --permanent`  
然后重启  
`firewall-cmd --reload`
